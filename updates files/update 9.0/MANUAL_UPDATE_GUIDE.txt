*****************************************************
Codychat manual update 9.0 instruction
****************************************************

if you do not have tryed the automatic update process please
refer to the other file UPDATE_GUIDE.txt instruction. The
current file is only for manual update process.

step 1
------

before attempting any update backup everything.

step 2
------

remove the files.zip from the 9.0 folder

step 3
------

upload the 9.0 folder to your chat "updates" folder

step 4
------
go into your admin panel in update section and run the update 9.0
from you. If you do not see anymore the update 9.0 it mean that
the update applyed correctly. if you receive a error please
contact our support.

step 5
------

unzip file.zip that you removed from the 9.0 folder and replace 
all files in your chat directory.


step 6
------

refresh all pages to load 9.0 new core files.


step 7
-------

go into your main chat and do a /clearcache in the main chat input then press enter

step 8
-------

enjoy Codychat 9.0
